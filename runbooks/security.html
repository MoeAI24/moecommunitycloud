Security Runbook

A practical checklist you can follow immediately.

Checklist

  • Enforce authentication (JWT/OAuth2) on all endpoints.
  • Use RBAC for tool access and admin actions.
  • Keep secrets in a secret manager (never in code).
  • Enable audit logs for prompts, tool calls, and deployments.
